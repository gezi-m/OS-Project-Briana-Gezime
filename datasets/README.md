# Datasets

## Paper 2 – Synchronization Benchmarks

Raw benchmark output from the experimental runs described in the paper.
Files are generated by `paper2_synchronization/benchmarks/run_benchmark_pc.py`
and `run_benchmark_dp.py`.

| File | Description |
|------|-------------|
| pc_N_sweep.csv | Producer-Consumer throughput vs N (M=2, K=2) |
| pc_M_sweep.csv | Producer-Consumer throughput vs M (N=10, K=2) |
| pc_K_sweep.csv | Producer-Consumer throughput vs K (N=10, M=2) |
| dp_all.csv     | Dining Philosophers throughput vs N (all solutions) |

## Paper 3 – MLFQ Simulation

Raw simulation output from the grid searches and scalability study.
Files are generated by `paper3_mlfq/simulation/run_all.py`.

| File | Description |
|------|-------------|
| grid_q1q2.csv     | 45-point Q1/Q2 grid search results |
| grid_l1l2.csv     | 30-point L1/L2 grid search results |
| grid_t_sweep.csv  | T% sweep from 0-100% |
| scalability.csv   | N vs metrics for M=50,100,200 |
| distributions.csv | Uniform vs Exponential vs Poisson comparison |
| mm1.csv           | M/M/1 model: L, Lq, W, Wq vs rho |
| mms.csv           | M/M/S model: metrics vs S (lambda=5, mu=3) |
